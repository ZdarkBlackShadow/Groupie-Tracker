About Me

How did you structure the project?

I structured the project to keep everything well-organized and separated to avoid confusion. The key phases were:

  • API research
  • API testing phase
  • Data retrieval in Golang
  • Data integration into templates
  • Implementation of features (search, filters, pagination)
  • Loading of all API data
  • Authentication system and collections
  • Frontend development (HTML, JS, and CSS)

How did you manage your time?

As I worked alone on this project, I set daily objectives:

  • Data integration into different templates
  • Development of features per page such as filters and pagination
  • Priority on essential functionalities
  • Testing and bug fixing

What was your documentation strategy?

I mainly documented myself through:

  • YouTube for detailed explanations and concrete examples
  • AI tools (DeepSeek, ChatGPT, Mistral) for code examples I could adapt
  • Stack Overflow for error resolution
  • Go's official documentation for functions and methods
